Rules

For 'Zone' we are using Mekton Zeta rules with a few adaptions from the hints we've gotten from Mekton Zero. The idea of archetypes is incorperated and all pc's are created using the available archetypes. Since all PC's are university students they all start with EDU at 3.

Archetype

In zone characters can be created using archetypes. These archetypes are templates to create a character with. Each archetype has an edge which can be used during sessions. Also they come with a set of starting skills and gear.

Roll a 1D10 then read your stats from the row you rolled. Or for every stat, roll and use the value from the row you rolled
